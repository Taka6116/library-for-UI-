Motion & Animation
モーション・アニメーション
KINETIC
Kinetic Typography
文字が単なるテキストではなく、動くビジュアル要素になる表現。各文字に波・回転・スケールのアニメーションを付与し、見出しをインパクトあるビジュアルに変える。MV・LP・ブランドサイトで効果的。
Liquid Motion
液体のようにぬるっと変形・遷移するエフェクト。border-radius の複数値アニメーションでBlobが有機的に変形。SVGフィルターや Framer Motion の keyframes でさらに液体感を強調できる。
Morphing Shapes
円・Blob・カードなどが滑らかに形状変化するアニメーション。clip-path や border-radius の補間で図形が変形。CSS @keyframes + cubic-bezier で物理感のある変化を表現できる。
Elastic Hover
ホバー時に要素がゴムのように伸縮するエフェクト。scaleX・scaleY を異なる値にして引き伸ばし感を演出。cubic-bezier(.34,1.56,.64,1) のスプリングイージングで自然な弾性を表現。
← hover でスプリング移動 →
Spring Easing
物理的なバネ感のあるイージング。cubic-bezier(.34,1.56,.64,1) で目標値を超えてから戻る挙動を再現。Framer Motion の spring 設定（stiffness/damping）でさらに精密な物理挙動も実装可能。
Overshoot Animation
目的位置を少し超えてから戻る、気持ちいい動き。scale(1.2) まで膨らんでから scale(1) に着地させる @keyframes で実現。クリック反応・モーダル出現など「確かな操作感」を演出するのに有効。

Design Tips · Gradients
グラデーション テクニック Tips
linear-gradient(127deg, …)
カスタム角度リニアグラデ
45°や90°を避けて127°など非対称な角度にする。動きが出てのっぺり感が消える。
radial-gradient(ellipse …)
ラジアルグロウセンター
中央から外に向かって透明になるラジアルグラデで光源感を演出。スポットライト効果。
コニックグラデーション
conic-gradient()で円形に色を展開。カラーホイールやデータ比率の可視化に使える。
Mesh Gradient
メッシュグラデーション
複数のradial-gradientを重ねてFigmaのMesh Gradient風に。::before ::afterで手軽に実現。
Animated Gradient
アニメーションブラシグラデ
background-size:600% + keyframeでpositionをシフト。CDN不要でSaaS感MAX。
Gradient Border
グラデーションボーダー
::before に inset:-1.5px + background:gradient でボーダー風に。border-imageより扱いやすい。
Gradient
TEXT GRADIENT
テキストグラデーション
background-clip:text + -webkit-text-fill-color:transparent。見出しに1行だけ使うのがコツ。
Gradient + Noise Texture
ノイズ + グラデーション
SVG feTurbulenceフィルターをグラデ上に重ねる。4%の粒子感でデジタルの硬さが和らぐ。
Glassmorphism
グラスモーフィズム
rgba背景 + backdrop-filter:blur(16px) + 白12%ボーダー。グラデ背景の上で使う。
Layered Depth
奥行きレイヤーグラデ
遠・中・近の3レイヤーでradial gradientを重ねる。背景に奥行きと光の雰囲気が生まれる。
Fade-to-dark overlay
ディレクショナルフェード
画像上のテキスト可読性を確保する「下→透明」グラデオーバーレイ。写真ヒーローで必須。
デュオトーン
2色グラデで写真やイラストに統一感を与えるテクニック。各カラム別色でも映える。
6-stop gradient
マルチストップグラデ
中間カラーストップを追加して曲線的な色変化を作る。2色だと直線的になりすぎる。
Vignette Effect
ビネット効果
中心→外周にかけて暗くするradial gradient。映画的な雰囲気と視線誘導の両方を実現。
Purple → Blue (135°)
Teal → Indigo (225°)
対角グラデカード
135°と225°の方向を組み合わせると自然なリズムが生まれる。ベンとグリッドと相性良い。
NEON GLOW
ネオングロウボーダー
box-shadow に同色を外→内の二重指定。発光しているように見えるエフェクト。
Stacked Transparent Layers
透明グラデ重ね
各方向から10%以下の薄いグラデを複数重ねる。単一グラデより複雑で有機的な背景に。
Inner Glow Card
インナーグロウカード
linear-gradient背景 + box-shadow(外部) + inset(内部ハイライト)の組み合わせ。
Hue-rotate Animation
ヒューシフトアニメーション
filter:hue-rotate() をkeyframeで360°回す。1つのグラデが全色を巡る省コストアニメ。
Prismatic Gradient
