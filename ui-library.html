Motion & Animation
モーション・アニメーション
KINETIC
Kinetic Typography
文字が単なるテキストではなく、動くビジュアル要素になる表現。各文字に波・回転・スケールのアニメーションを付与し、見出しをインパクトあるビジュアルに変える。MV・LP・ブランドサイトで効果的。
Liquid Motion
液体のようにぬるっと変形・遷移するエフェクト。border-radius の複数値アニメーションでBlobが有機的に変形。SVGフィルターや Framer Motion の keyframes でさらに液体感を強調できる。
Morphing Shapes
円・Blob・カードなどが滑らかに形状変化するアニメーション。clip-path や border-radius の補間で図形が変形。CSS @keyframes + cubic-bezier で物理感のある変化を表現できる。
Elastic Hover
ホバー時に要素がゴムのように伸縮するエフェクト。scaleX・scaleY を異なる値にして引き伸ばし感を演出。cubic-bezier(.34,1.56,.64,1) のスプリングイージングで自然な弾性を表現。
← hover でスプリング移動 →
Spring Easing
物理的なバネ感のあるイージング。cubic-bezier(.34,1.56,.64,1) で目標値を超えてから戻る挙動を再現。Framer Motion の spring 設定（stiffness/damping）でさらに精密な物理挙動も実装可能。
Overshoot Animation
目的位置を少し超えてから戻る、気持ちいい動き。scale(1.2) まで膨らんでから scale(1) に着地させる @keyframes で実現。クリック反応・モーダル出現など「確かな操作感」を演出するのに有効。

Fluid Gradient
Fluid Gradient Animation
グラデーションが液体のように流動する背景。background-position を background-size 400% の範囲でアニメーションさせることで色が流れる表現を実現。疑似的な動的グラデーション。

Light Sweep

Iridescent Gradient

CHROME UI
Chrome / Metallic UI

Scale In
中央からスケールアップして現れる、最も汎用的なモーダルアニメーション。cubic-bezier のスプリングで少し弾む。
Scale In Modal
中央から scale(0.8→1) でポップインするモーダル。最も汎用的で「確かな出現感」を与える。opacity と transform を同時に transition させ、cubic-bezier(.34,1.56,.64,1) で少しオーバーシュートさせるのが現代的。
Slide Up
下から上にスライドして登場するモーダル。モバイルのボトムシートと異なり、中央に配置される。確認ダイアログ・フォームに適している。
Slide Up Modal
translateY(50px→0) + opacity でモーダルが下から浮かび上がるアニメーション。重力感があり「画面から生まれた」印象を与える。オーバーレイは blur + opacity のみで処理し、モーダル本体だけ動かすのがポイント。
Elastic Pop!
scale(0) + rotate(-8deg) から弾むように登場。通知・バッジ・アラートなど「注目させたい」モーダルに最適なアニメーション。
Elastic Pop Modal
scale(0) rotate(-8deg) から .55s cubic-bezier(.34,1.56,.64,1) で弾み出すモーダル。ゲーミフィケーション・アチーブメント・エラー通知など、感情的インパクトが必要な場面で使う。Framer Motion の spring type に相当。

Blur Reveal Modal
